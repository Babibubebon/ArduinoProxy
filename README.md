ArduinoProxy
============

ArduinoとFirefox OSシミュレータをつなぎます

##必要なもの
1. Arduino IDE
2. Arduno
3. Processing
4. Firefox 
5. Firefox OS シミュレータ

##使い方

1. Processingを起動して, arduinoFxOSProxy/arduinoFxOSProxy.pde を開く
2. Arduinoに arduinoServer/arduinoServer.ino を書き込む
3. Firefox OS シミュレータを起動する
4. あとはmozTCPSocketで楽しむ
